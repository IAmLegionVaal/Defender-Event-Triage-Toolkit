# Defender Event Triage Toolkit

A read-only PowerShell toolkit for Microsoft Defender health and event review.

## Features

- Defender service and preference summary
- Signature and engine version reporting
- Recent Defender operational events
- CSV, JSON, and HTML reports

## Run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\Defender_Event_Triage_Toolkit.ps1
```

## Safety

Read-only reporting only. No Defender settings are changed.
